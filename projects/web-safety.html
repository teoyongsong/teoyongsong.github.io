Web Safety Checker (API-first)

Security tooling project — Mar 2026

An API-first web safety tool that checks URL safety, domain email hygiene, and host network posture, with optional combined analysis and downloadable PDF reports.

Overview

The project exposes dedicated APIs for web checks, email checks, network checks, and an all-in-one endpoint that returns sectioned results. The frontend calls the backend API directly, and users can generate a full PDF report after each assessment.

Main API endpoints

  • POST /check/web — URL and link safety checks.
  • POST /check/email — email hygiene checks (MX/SPF/DMARC).
  • POST /check/network — host reachability and TLS posture checks.
  • POST /check/all — combined web + email + network results.
  • POST /report/pdf — downloadable full report with recommendations.

Highlights

  • Supports Web/Email/Network modes from one interface.
  • API-first architecture for easier integration and automation.
  • PDF report export for sharing findings and recommendations.
  • Health endpoint for service monitoring.

Run locally

  • Start API server: python3 backend_api.py
  • Backend listens on: http://127.0.0.1:8787
  • Health check: GET /health
